Questions fréquentes

Comment signaler une erreur de donnée ?

Précisez le jeu (nom, année), la valeur observée et la valeur attendue. Nous vérifions auprès des directions régionales.

Où obtenir une clé API ?

Consultez la page API et faites une demande ; une clé de test est fournie pour 1 000 requêtes/jour.

Peut-on partager les données publiquement ?

Oui, sous réserve de citation de la source Ministère de l'Agriculture et de l'Élevage.
